Conversão de Base via Divisão Sucessiva

Sistemas de Numeração

Divida sucessivamente o valor de base dez para a conversão pretendida, depois é só ir voltando e anotando os restos da divisão a cada etapa, p. ex. 72 para a base três:


	72/3 = 24, resto 0 | 24/3 = 8, resto 0 | 8/3 = 2, resto 2 | Fim.
	Resultado: 2200
	
	Verificação: 2x3³+2x3²+0x3¹+0x3 = 72
